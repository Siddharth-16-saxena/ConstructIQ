# ConstructIQ

ConstructIQ is a smart construction operations website for a mid-market project-control platform. It includes a responsive product page, interactive what-if dashboard, live alert mockups, mobile field app preview, pricing, and an ROI calculator.

## Run Locally

Open `index.html` in a browser.

## Deploy

This repository includes a GitHub Pages workflow at `.github/workflows/pages.yml`. Push the `main` branch to GitHub, then enable Pages with GitHub Actions as the source if prompted.
